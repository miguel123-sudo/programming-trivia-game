# Programming Trivia Game 🎮

Un videojuego educativo de trivia de programación multijugador para preparatoria.

## Características

✅ Sistema de autenticación completo (Email, Google, Facebook, Teléfono, Invitado)
✅ Panel de usuario con perfil personalizado
✅ 5 categorías de programación
✅ Batallas 1v1 en tiempo real
✅ Preguntas aleatorias con timer
✅ Sistema de puntos y vida
✅ Ranking mundial
✅ Fondos animados
✅ Seguimiento de progreso

## Instalación

```bash
git clone https://github.com/miguel123-sudo/programming-trivia-game.git
cd programming-trivia-game
pip install -r requirements.txt
python main.py
```

## Requisitos

- Python 3.8+
- Pygame
- SQLite3

## Estructura del Proyecto

```
programming-trivia-game/
├── main.py                 # Archivo principal
├── database.py             # Gestión de base de datos
├── ui.py                   # Interfaz de usuario
├── game.py                 # Lógica del juego
├── questions.py            # Base de preguntas
├── config.py               # Configuración
├── assets/
│   ├── backgrounds/        # Fondos animados
│   ├── sounds/             # Efectos de sonido
│   └── images/             # Sprites e iconos
└── requirements.txt        # Dependencias
```

## Controles

- **Ratón**: Navegar por menús
- **Teclado**: Seleccionar respuestas (1, 2, 3, 4)
- **ESC**: Volver atrás

## Categorías

1. Fundamentos de Programación
2. Variables y Tipos de Datos
3. Funciones
4. Programación Orientada a Objetos (POO)
5. Algoritmos
